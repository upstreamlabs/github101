## 4. Git命令使用过程总结


- git clone: 克隆远程仓库到本地
- git add: 添加文件到暂存区
- git commit: 提交更改到本地仓库
- git push: 推送更改到远程仓库
- git pull: 从远程仓库拉取最新更改
- git merge: 合并分支，解决冲突
- git fetch: 从远程仓库的最新提交下载到本地仓库，但不会改变你当前工作的分支
- git rebase: 变基，类似合并，但可以是的提交历史更清晰，但是算是破坏性更改
